# MOONCRAFT

[![Build](https://github.com/melinakrzemowa/mooncraft/actions/workflows/deploy.yml/badge.svg)](https://github.com/melinakrzemowa/mooncraft/actions/workflows/deploy.yml)

check node version (16.0.0^)  
install packages: `yarn install`  
run project: `yarn start`  
it is running on: http://localhost:1969/

EXPLORE THE MOON!

build: `yarn build`  
game files will be in `dist` folder

Map creating: "Tiled" from https://www.mapeditor.org/
